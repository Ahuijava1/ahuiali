# 体重记录需求
> 记录每天体重，达到体重记录可视化的效果，为用户提供更好的掌握体重信息的功能。

记录包括用户id，体重，计量单位，记录日期，类型（早饭，午饭），创建时间，修改时间。
1. 增加记录
    用户新增一条数据
2. 修改记录
    用户修改一条记录
3. 删除记录
    用户删除一条记录
    用户删除今天记录*
    用户清空某段时间记录*
    用户清空所有记录*
4. 查询记录
    默认查询今日记录
    查询某段时间记录
    记录显示最大值、最小值、平均值、方差*
